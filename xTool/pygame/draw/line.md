# line()

```python
# 在Surface上绘制一条黑色直线
pygame.draw.line(s, (0, 0, 0), (250, 250), (250 + 200, 250))
```

参数解释：

* s - 目标Surface对象，即之前创建的透明画布
* (0, 0, 0) - 颜色值，RGB格式，黑色
* (250, 250) - 起点坐标，屏幕中心点
* (250 + 200, 250) - 终点坐标，从中心向右200像素的位置

功能说明：

* 绘制一条从屏幕中心(250,250)到右侧(450,250)的水平直线
* 直线长度为200像素
* 在透明Surface上绘制，可以与其他图形叠加显示
